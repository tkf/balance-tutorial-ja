.. _def-J:

結合の分布 [#]_

.. math::

   \Prob \{ J_{kl}^{ij} = J_{kl}/\sqrt K \} = K/N_l

   \Prob \{ J_{kl}^{ij} = 0 \} = 1 - K/N_l

.. [#] 原著 [vanVreeswijk1998]_ には確率は :math:`K/N_k`
   と書いてあるがこれは間違い.
